# CO2 Tracker
Track your carbon emissions with this simple app.


## 🌍 Overview 

Co2 tracker helps users monitor their carbon emissions by providing accurate estimates for common daily activities. 
Our goal is to make environmental awareness easy and accessible through a simple app.


## 🔧 Features

- Submit activities and get CO₂ emissions.
- View activity history and total emissions.

### Supported activities
- Electricity consumption: User needs to provide the power consumption of the appliance (kWH) and the time of use.


## 🛠️ Tech Stack

- **Backend**: Django (Python 3.11)
- **Database**: PostgreSQL
- **Containerization**: Docker & Docker Compose
